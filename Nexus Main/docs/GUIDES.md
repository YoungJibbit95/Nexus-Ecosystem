# Nexus v5.0 Guides

## Dashboard

- Open `Layout bearbeiten` to enter the layout editor.
- Drag widgets to the snap board or directly in the grid.
- Fine-tune with:
  - width: `1w` / `2w`
  - column: `C1` / `C2`
  - row nudges: `R-` / `R+`
- Use `Reset Layout` to restore defaults.

## Notes

- Modes: `edit`, `split`, `preview`
- Magic widgets supported in markdown code blocks:
  - `nexus-list`
  - `nexus-alert`
  - `nexus-progress`
  - `nexus-timeline`
  - `nexus-grid`
  - `nexus-card`

## Code

- Monaco-based editor with multi-language support
- `Ctrl+Enter` for JS/TS execution
- HTML/CSS live preview in split/preview modes
- JSON validation and formatting support

## Tasks

- Kanban status flow: `todo`, `doing`, `done`
- Subtasks and notes inside task details modal
- Priority and due-date based visual cues

## Reminders

- Views/filters: `upcoming`, `soon`, `overdue`, `done`, `all`
- Overdue quick actions:
  - snooze `+15m`
  - snooze `+1h`
  - `clear done`
- Toast workflow: dismiss or snooze from notification card

## Canvas

- Magic Builder templates:
  - `mindmap`
  - `roadmap`
  - `sprint`
  - `risk-matrix`
  - `decision-flow`
  - `ai-project`
- Auto-layout modes:
  - `mindmap`
  - `timeline`
  - `board`
- AI Project template:
  - prompt-driven plan generation
  - depth levels: `light`, `balanced`, `deep`
- Double-click canvas to quick-add nodes.

## Files

- Use workspaces to group items by context.
- Switch between `grid` and `list` view modes.
- Assign notes/code/tasks/reminders to custom workspaces.

## DevTools

- Visual CSS/Tailwind element builder
- Utility calculator groups:
  - spacing
  - color
  - typography
  - layout
  - animation

## Settings

- Theme, glass, glow, background, layout, animation, and editor tuning
- Changes are live-applied without manual save

## Terminal + Spotlight

- Terminal opens centered at bottom.
- Spotlight: `Shift` x2 or `Cmd/Ctrl + K`.
- Useful commands:
  - `help`
  - `views`
  - `goto <view>`
  - `search <query>`
  - `canvas template ai <name>`
  - `macro start <name>`, `macro stop`, `macro run <name>`
  - `undo`, `redo`
