## 🎯 What is Nexus?

A **multi-app ecosystem** with shared runtime, control plane and live sync.

## 🧩 Components

- Nexus Main (Electron)
- Nexus Mobile (Capacitor)
- Nexus Code
- Nexus Code Mobile
- Nexus API Client (nexus-core)
- Control Plane (private)

## 🏗️ Architecture

Apps → Runtime → Control Client → Control Plane

## 🔄 Live Sync

- Feature sync across apps
- Layout adaptation
- Capability-based updates

## 🚀 Quick Start

```bash
git clone https://github.com/YoungJibbit95/Nexus-Ecosystem.git
cd Nexus-Ecosystem
npm run setup
npm run build
```

## 🛠️ Dev Commands

```bash
npm run dev:all
npm run dev:all:with-control-ui
npm run dev:main
npm run dev:mobile:android
npm run dev:mobile:ios
npm run dev:code
npm run build
npm run verify:ecosystem
npm run doctor:release
```

## ⚙️ Control Plane

- Hosted backend
- Auth / Config / Policies
- UI deployable separately

## 🔐 Security

- Role-based access
- Device verification
- HMAC signatures
- Anti-replay
- Audit logs

## 📦 Build System

```txt
build/
├── Nexus Main
├── Nexus Mobile
├── Nexus Code
├── Nexus Control
├── API Client
└── assets
```

## 📋 Workflow

1. Create issue
2. Build feature
3. Verify
4. PR
5. Deploy

## 🧯 Troubleshooting

- Check API URL
- Check ENV
- Check device verification
- Check origins
