# Support

Use GitHub Issues for public client bugs, documentation problems and feature requests.

Do not post secrets, tokens, private account data, payment details, private cloud logs or infrastructure details in public issues.

## Security Issues

Report vulnerabilities privately through GitHub Security Advisories. See [SECURITY.md](./SECURITY.md).

## Cloud Account Issues

Account, billing, sync and cloud entitlement issues are handled through Nexus support channels outside this public repository. Public issues should avoid private account details.
