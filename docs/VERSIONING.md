# Versioning

Nexus uses compatible release groups across multiple clients.

## Rules

- The root package is a private monorepo workspace.
- App versions may differ.
- Releases group compatible app builds.
- Nexus Cloud compatibility is server-managed.
- Public docs should not describe private cloud route or deployment internals.

## Current Public Naming

- Nexus Main
- Nexus Mobile
- Nexus Code
- Nexus Code Mobile
- `@nexus/core`
