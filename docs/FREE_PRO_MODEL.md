# Free / Pro Model

Nexus is designed to be useful as a local-first client while keeping cloud-backed capabilities account-bound.

## Free

- Local notes
- Local tasks
- Local reminders
- Local canvas boards
- Local files and workspace organization
- Basic local code workflows
- Public client builds and documentation

## Pro / Nexus Cloud

- Cloud sync
- Account-backed backups
- AI/Flux features
- Multi-device continuity
- Sharing and team workflows
- Higher account limits
- Cloud-assisted recovery and handoff features

## Enforcement

Client-side UI states are not security controls. Pro, sync, payment, AI usage, team features and cloud limits must be enforced by Nexus Cloud.

Public docs should describe the model at this level and avoid private implementation details.
